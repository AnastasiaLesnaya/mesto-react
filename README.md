# Проект: Место

### Что это?

* Сайт, где есть возможность делиться прекрасными фото различных мест и ставить друг другу лайки.

**Технологии**

* Для создания использовались: flexbox, grid, javascript, API и др.
* Проект портирован на "Реакт".
